# Visão Geral

Este projeto tem como objetivo desenvolver uma aplicação de realidade aumentada/realidade mista voltada para a criação e organização de ambientes virtuais, permitindo ao usuário adicionar, mover e posicionar objetos em um espaço real.

A aplicação é especialmente útil para áreas como arquitetura, design de interiores e planejamento de espaços, oferecendo uma forma intuitiva de visualizar a disposição de móveis e objetos em escala real.

O usuário poderá visualizar, diretamente na tela VR do Meta Quest, uma lista lateral de objetos, realizar drag and drop e posicioná-los no ambiente com precisão, avaliando proporções, layout e ocupação do espaço.
