# Ciclo Motivação-Ação-Feedback 

Para criar uma solução que seja interessante e até viciante para o usuário, é preciso pensar em maneiras de engajar os utilizadores. 

Para isso, uma das maneiras pode ser através de ciclos de engajamento, em que o usuário recebe motivações que estimulam-os a tomarem ações que vão gerar uma resposta, que indica se a ação gerou ou não o resultado esperado, motivando o usuário a seguir. 

Esta abordagem vem da criação de jogos, em que é imprescindível engajar o jogador para ele continuar jogando o jogo por mais tempo.

Dessa forma, foi feito um ciclo motivação-ação-feedback considerando o projeto e todo o contexto em que ele está inserido: 

### Motivação

**Visualizar o inexplorado**: Necessidade de testar, validar e experimentar composições espaciais de forma imersiva, sem custos físicos ou complexidade técnica excessiva.

### Ação

**Modelagem Intuitiva**: O usuário seleciona objetos no menu e utiliza controladores VR para "Drag and Drop", reposicionando e rotacionando itens no espaço e com a possibilidade de alterar cores.

### Feedback

**Materialização Instantânea**: O ambiente se transforma em tempo real. O usuário vê a composição finalizada, validando a estética e a funcionalidade do layout proposto.
