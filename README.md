# Visão Geral

Este projeto tem como objetivo desenvolver uma aplicação de realidade aumentada/realidade mista voltada para a criação e organização de ambientes virtuais, permitindo ao usuário adicionar, mover e posicionar objetos em um espaço real.

A aplicação é especialmente útil para áreas como arquitetura, design de interiores e planejamento de espaços, oferecendo uma forma intuitiva de visualizar a disposição de móveis e objetos em escala real.

O usuário poderá visualizar, diretamente na tela VR do Meta Quest, uma lista lateral de objetos, realizar drag and drop e posicioná-los no ambiente com precisão, avaliando proporções, layout e ocupação do espaço.

# Objetivos Principais

Criar um ambiente imersivo onde seja possível adicionar objetos 3D no espaço.
Permitir interação intuitiva através de drag and drop usando controladores VR.
Possibilitar ao usuário reorganizar facilmente os objetos virtuais dentro do ambiente.
Auxiliar arquitetos e designers a prever e testar diferentes composições espaciais.
Manter liberdade tecnológica para definição de frameworks, engines e bibliotecas.

# Funcionalidades Planejadas
- Interface no Meta Quest:
  - Lista de objetos disponíveis exibida no canto da tela (menu lateral).
  - Miniaturas 3D dos objetos para facilitar identificação.
  - Seleção e manipulação de objetos através dos controladores VR.

- Interação com Objetos
  - Drag and drop dos objetos para o ambiente.
  - Rotação, escala e reposicionamento.
  - Sistema de grade opcional para alinhamento preciso.
  - Salvamento da composição criada.

- Ambiente
  - Ambiente virtual base minimalista OU uso do espaço real via passthrough (dependendo da tecnologia adotada).
  - Planejamento para posterior inclusão de texturas, iluminação e materiais aprimorados.
